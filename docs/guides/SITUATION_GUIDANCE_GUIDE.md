# Situation Guidance Tool 가이드

## 🎯 목적

사용자가 법에 의해 받는 스트레스를 덜어주기 위해, **상황 기반 종합 법적 가이드**를 제공합니다.

## 🔄 작동 원리

### 1. 상황 분석

사용자의 법적 상황을 입력받아:

- **법적 영역 자동 감지** (개인정보, 노동, 세금, 부동산 등)
- **핵심 용어 추출** (법령명, 기관명, 날짜, 금액 등)

### 2. 통합 검색

다음 정보를 **자동으로 모두 검색**:

- 관련 법령
- 유사 판례
- 관련 기관 해석
- 행정심판 사례
- 위원회 결정문

### 3. 단계별 가이드 제공

사용자에게 **실행 가능한 단계별 가이드** 제공

## 📝 사용 예시

### 예시 1: 노동 문제

```
사용자: "회사에서 해고당했는데 퇴직금을 받지 못했어요"

→ 자동 감지: 노동 영역
→ 관련 법령: 근로기준법, 고용보험법
→ 유사 판례: 해고 관련 판례 검색
→ 기관 해석: 고용노동부 해석 확인
→ 행정심판: 노동위원회 결정문 검색
→ 가이드:
  1. 근로기준법 확인
  2. 유사 판례 검토
  3. 노동위원회에 진정 제기 고려
```

### 예시 2: 개인정보 유출

```
사용자: "개인정보가 유출되었는데 어떻게 해야 하나요"

→ 자동 감지: 개인정보 영역
→ 관련 법령: 개인정보보호법, 정보통신망법
→ 유사 판례: 개인정보 유출 판례
→ 기관 해석: 개인정보보호위원회 해석
→ 위원회 결정: 개인정보보호위원회 결정문
→ 가이드:
  1. 개인정보보호법 확인
  2. 개인정보보호위원회에 신고
  3. 유사 사례 참고
```

### 예시 3: 세금 문제

```
사용자: "세금 부과가 부당하다고 생각합니다"

→ 자동 감지: 세금 영역
→ 관련 법령: 소득세법, 부가가치세법
→ 유사 판례: 세금 관련 판례
→ 기관 해석: 국세청 해석
→ 특별심판: 조세심판원 재결례
→ 가이드:
  1. 관련 세법 확인
  2. 조세심판원에 심판 청구 고려
  3. 유사 사례 참고
```

## 🛠️ LLM 사용 가이드

### LLM이 해야 할 일

1. **사용자 상황을 그대로 전달**

   ```json
   {
     "name": "situation_guidance_tool",
     "arguments": {
       "situation": "회사에서 해고당했는데 퇴직금을 받지 못했어요"
     }
   }
   ```

2. **결과를 사용자 친화적으로 해석**
   - 단계별 가이드를 자연어로 설명
   - 관련 법령, 판례, 해석을 요약
   - 다음 단계 제안

### 응답 구조

```json
{
  "situation": "사용자 상황",
  "detected_domains": ["노동"],
  "key_terms": {
    "laws": ["근로기준법"],
    "agencies": ["고용노동부"],
    "keywords": ["해고", "퇴직금"]
  },
  "laws": {
    "근로기준법": {...}
  },
  "precedents": {...},
  "interpretations": {...},
  "administrative_appeals": {...},
  "committee_decisions": {...},
  "guidance": {
    "steps": [
      {
        "step": 1,
        "title": "관련 법령 확인",
        "description": "...",
        "action": "..."
      }
    ]
  },
  "summary": "법적 영역: 노동 | 관련 법령 1개 발견 | 유사 판례 5개 발견"
}
```

## 🎨 특징

### 1. **통합성**

- 여러 법령, 기관, 판례를 한 번에 검색
- 사용자가 따로 찾을 필요 없음

### 2. **지능성**

- 상황에서 자동으로 법적 영역 감지
- 관련 기관 자동 매핑

### 3. **실용성**

- 단계별 실행 가능한 가이드 제공
- 유사 사례를 통한 참고 자료 제공

### 4. **사용자 중심**

- 법적 스트레스 완화
- 복잡한 법적 정보를 이해하기 쉽게 제공

## 📊 지원하는 법적 영역
