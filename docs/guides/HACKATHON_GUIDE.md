# 해커톤 제출 가이드

변호사 MCP 서버를 해커톤에 제출하기 위한 가이드입니다.

## 📋 제출 전 체크리스트

### 1. MCP 스펙 준수 확인

- [ ] MCP 스펙 버전 2025-03-26 이상 준수
- [ ] Streamable HTTP 방식으로 구현
- [ ] Remote 서버로 공개 URL 제공
- [ ] Stateless 서버 설계 (no session)
- [ ] MCP Inspector로 사전 점검 완료

**참고**: [MCP 심사 가이드](./MCP_COMPLIANCE_RULES.md)의 "MCP 서버 생성 가이드" 섹션 확인

---

### 2. 툴 요구사항 확인

- [ ] 최소 1개 이상의 툴 포함 ✅ (현재 7개)
- [ ] 툴 개수 3~20개 권장 범위 내 ✅ (현재 7개)
- [ ] 모든 툴 사전 테스트 완료
- [ ] 응답 속도 및 안정성 확인
- [ ] Response 크기 24k 이하 확인

**현재 툴 목록**: [TOOLS_LIST.md](./TOOLS_LIST.md) 참고

### 3. 툴 품질 확인

- [ ] 사용자가 어떤 질문을 할 때 사용하는지 명시

#### 3.2 Name 품질
- [ ] 툴 이름이 간결하고 기능을 대표함
- [ ] 소문자, 언더스코어 사용 (예: `search_law_tool`)
- [ ] "kakao" prefix/suffix 미사용

#### 3.3 동작 확인
- [ ] 모든 툴이 실제로 동작함
- [ ] 에러 케이스 처리 확인
- [ ] API 키 없을 때 적절한 에러 메시지 반환

### 4. 정책 준수 확인

- [ ] 개인정보 수집/전송 최소화
- [ ] 금지된 개인정보 요구/전송 없음
- [ ] 상업적 행위 없음

## 📝 제출 정보 작성

### MCP 서버 정보

- **이름**: LexGuard MCP (또는 변호사 MCP)
- **설명**: 일반인들이 AI를 통해 법률 정보를 쉽게 조회할 수 있도록 도와주는 MCP 서버입니다. 국가법령정보센터의 159개 API를 활용하여 법령 검색, 조문 조회, 판례 검색 등의 기능을 제공합니다.
- **카테고리**: 법률 / 법제 / 공공서비스

### 주요 기능

1. **법령 검색**: 법령명이나 키워드로 관련 법령 검색
2. **법령 조문 조회**: 특정 법령의 조문 전체 또는 단일 조문 조회
3. **법령 상세 정보**: 법령의 기본 정보, 제정일, 개정 이력 조회
4. **법령명 목록**: 전체 법령명 목록 조회 및 필터링
5. **API 목록 조회**: 사용 가능한 159개 API 목록 확인

### 사용 사례

- "형법 제1조 내용 알려줘"
- "개인정보보호법 관련 법령 검색해줘"
- "근로기준법에서 해고 관련 조문 찾아줘"
- "손해배상 관련 판례 검색해줘"
